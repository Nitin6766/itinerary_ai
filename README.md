# 🌍 AI Travel Itinerary Generator

An AI-powered travel planner that generates day-wise itineraries based on your budget, location, and trip duration.

---

## 🚀 Features

- 🗺️ Location-based travel plans
- 💰 Budget-friendly suggestions
- 📅 Day-wise itinerary
- 🍽️ Food recommendations
- 🤖 AI-generated travel guide

---

## 🛠️ Tech Stack

### Frontend

- React (Vite + TypeScript)
- Axios

### Backend

- FastAPI (Python)
- Groq API (LLaMA model)

---

## 📦 Installation

### 🔹 Backend

```bash
cd backend
pip install fastapi uvicorn requests
uvicorn main:app --reload
```
